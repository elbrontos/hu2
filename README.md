# Hausübung 2 – IT-Security (Aufgabe 3c)

Dieses Repository enthält die Lösung zu Aufgabe 3c der Hausübung 2 im Modul
„Einführung in IT-Security“.

## Inhalt
- RSA-Schlüsselerzeugung (Public Key / Private Key)
- SHA-256 Hashfunktion
- RSA-Verschlüsselung und -Entschlüsselung
- Digitale Signaturen (Signieren und Verifizieren)
- Automatische Tests (inkl. Manipulationstest)

## Datei
- hu2_aufgabe3c.py

## Ausführung
Das Programm kann mit Python 3 ausgeführt werden:

python3 hu2_aufgabe3c.py

## Hinweise
Die Implementierung dient ausschließlich Übungszwecken.
Es wird bewusst kein Padding-Verfahren (z. B. OAEP oder PSS) verwendet.
Für reale Anwendungen wären größere Schlüssellängen (≥ 2048 Bit) erforderlich.
